# Info Siantar

Kumpulan API artikel berita seputar kota Pematang Siantar.

- /antarasumut
- /armadanews
- /bentengsiantar
- /bidiknewstoday
- /dekrit
- /harianmetro
- /hetanews
- /idntimes
- /jurnalx
- /kliktodaynews
- /lintangnews
- /lintaspublik
- /mistar
- /newscorner
- /rmolsumut
- /sbnpro
- /sib
- /tribunmedan

Cara menjalankan:

```
npm install
npm run infosiantar
```

Contoh:

```
GET http://localhost/hetanews
```
